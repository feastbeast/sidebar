## Usage

To start server:
```sh
npm run start
```

To run webpack:
```sh
npm run build
```

## Requirements

- Node 6.13.0
- ExpressJS
- React
- React-DOM
- Webpack
- Babel CLI
- MongoDB
- Mongoose


## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```
